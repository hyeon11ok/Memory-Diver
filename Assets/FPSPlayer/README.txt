**1인칭 플레이어**

**주의사항**
- BaseCondition import 필요
- UI FrameWork import 필요
- 점프를 위해 Ground 레이어 설정 필요
- InputSystem 설정 필요(유니티 6.1버전 이후의 기본 세팅 사용중)
- Player 프리펩은 모든 클래스 기본 설정을 해놓은 상태로 만듬, 추가로 상황에 맞게 수정
- 캐릭터 모델은 프리펩 하위에 생성

**Player**
- 플레이어 관련 클래스를 보관하는 클래스
- 외부에서 플레이어의 다른 클래스에 접근하는 것을 원활하게 하기 위함

**InputHandler**
- 플레이어의 입력을 받아 보관하는 클래스
- 기본적인 입력(이동, 회전, 점프, 달리기)만 구현, 필요한 기능은 추가 구현 필요

**PlayerCondition**
- 유닛의 상태값을 관리하는 클래스인 BaseCondition을 상속받아 구현
- 스테미너를 사용한다는 전제하에 구현, 불필요하면 수정 필요
- 상황에 맞게 추가로 필요한 상태값은 인스펙터에서 설정 가능

**PlayerController**
- 플레이어 조작 관련 기능 구현
- 이동 속도, 점프 파워, 마우스 감도 등 존재

**Interaction**
- 플레이어가 상호작용할 수 있는 오브젝트를 감지하고 상호작용하는 기능 구현
- 상호작용 가능한 오브젝트는 IInteractable을 상속받아 구현
